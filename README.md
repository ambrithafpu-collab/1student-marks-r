# 📊 Student Marks Calculator (R)

A beginner-friendly R program that calculates total marks, percentage, and grade for students based on their marks in four subjects.

---

## 🚀 Project Overview

This R script demonstrates basic data handling and simple calculations in R using a data frame.  
It reads marks for students, computes total marks and percentage, and assigns a grade.

---


---

## 🧠 How It Works

The script:

1. Defines a table of students with marks in four subjects.
2. Calculates total marks.
3. Computes percentage.
4. Assigns a grade based on percentage.
5. Prints the results.

---

## 💻 How to Run

1. Download or clone the repository.
2. Open **student_marks.R** in RStudio or any R environment.
3. Run the script.
4. See the output in the R console.

---

## ⭐ Example Output

When the script runs successfully, you will see something like:

```
     Name English Maths Science Social Total Percentage Grade
1 Ambritha      85    90      88     92   355     88.75      A
```
