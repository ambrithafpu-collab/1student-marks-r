# Student Marks Calculator (R)

This project is a simple R program that creates a table of student marks in four subjects:

- English  
- Maths  
- Science  
- Social

It then calculates:

✔ Total marks  
✔ Percentage

---

## 📄 Files in this Repository

- `student_marks.R` — R script with the code  
- `.gitignore` — tells Git which files not to track  
- `README.md` — this file  
- `LICENSE` — open-source license

---

## 📌 How to Use

1. Open this repository in GitHub or download it to your computer.
2. Open the file `student_marks.R` in **RStudio**.
3. Run the script.
4. View the output in the RStudio console.
